Smartarget Information Message

Show important updates and information on your site

  • Improve conversion
  • Improve users trust
  • Free
  • No code
  • 3 min integration

Smartarget developed a plugin for Odoo so you can communicate information to users in a visible way, yet not intrusive.

The Information Message will be shown to users on all pages, both on desktop and mobile, and will allow you to communicate with them relevant information that you believe is important for them to know.

Check the Demo page to see how it work: https://smartarget.online/demo.html?app=flash_cards

How to configure

  • 1. Install module
  • 2. Go to Settings/Website
  • 3. Fill in Smartarget User ID or Information Message settings (you can get Smartarget User ID on https://smartarget.online)
