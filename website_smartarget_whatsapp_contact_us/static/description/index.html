Smartarget Whatsapp - Contact Us

Allow customers to contact you using Whatsapp

  • Improve conversion
  • Improve users trust
  • Free
  • No code
  • 3 min integration

Let's make it easier for your customers to contact you! Probably most of your users are using Whatsapp, so its time to give them the option to contact you by using Whatsapp. With 'Whatsapp - Contact Us' you can add a small icon on the bottom-right corner with you phone number and short message as "Contact us" or "message us" etc. When user click on the icon - new tab will be pop with Whatsapp for web and chat window with you.

How to configure

  • 1. Install module
  • 2. Go to Settings/Website
  • 3. Fill in Smartarget User ID or Phone Number (you can get Smartarget User ID on https://smartarget.online)
