Keep users from leaving your website!

Exit Popup allowing you to show a message to users who move their mouse courser in attention to close your website tab. With Exit Popup you can design your popup message and call-to-action to attrack the user and prevent his leaving.

Check the Demo page to see how it work: https://smartarget.online/demo.html?app=exit_preventor

How to configure

  • 1. Install module
  • 2. Go to Settings/Website
  • 3. Fill in Smartarget User ID or Exit Popup settings (you can get Smartarget User ID on https://smartarget.online)
